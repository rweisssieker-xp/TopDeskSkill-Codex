# Commercial Model

The TOPdesk Skill Suite plugin is free open-source software. The commercial value is not a software license sale; it is the accelerated delivery of reporting, AI/KI governance, lifecycle analytics, migration support, and operational enablement.

## Free Components

- Codex plugin package.
- Bundled TOPdesk skills.
- Local MCP helper scripts.
- Power BI templates, DAX snippets, and Power Query patterns.
- Demo lifecycle CSV files.
- AI/KI governance templates and evaluation scaffolds.
- USP battlecards, proof-of-value templates, and ROI templates.

## Billable Or Separately Owned Components

- TOPdesk tenant access and TOPdesk licenses.
- Power BI licenses, gateway, capacity, and workspace administration.
- Customer-specific data mapping and model design.
- Implementation of scheduled snapshots and operational monitoring.
- AI provider, model, gateway, or enterprise platform usage.
- Training, rollout, adoption, support, and managed service.
- Custom integrations, security review, and compliance documentation.

## Offer Packages

| Package | Purpose | Typical Output |
| --- | --- | --- |
| Free Evaluation | Let a customer inspect the plugin and run demo assets. | Installed plugin, demo prompts, demo lifecycle CSV review. |
| Five-Day Proof | Prove value with a bounded dataset and decision-ready findings. | Data profile, lifecycle bottlenecks, KPI shortlist, PoV readout. |
| Reporting Foundation | Build a controlled Power BI model from TOPdesk data. | Semantic model, core KPIs, refresh approach, data-quality page. |
| Lifecycle Analytics | Make status and assignment history auditable. | Snapshot pipeline, transition facts, dwell-time KPIs, reconciliation checks. |
| AI/KI Evaluation | Test AI use cases without uncontrolled automation. | Use-case backlog, prompt/eval pack, monitoring KPIs, governance decision. |
| Managed Improvement | Keep reporting, data quality, and AI monitoring useful over time. | Monthly findings, cleanup backlog, KPI tuning, release notes. |

## Positioning Lines

- The tool is free; the value is faster insight, safer delivery, and less custom rework.
- No extra analytics add-on is required to start.
- The customer keeps the assets and can inspect, fork, or adapt them.
- The first business case should measure implementation effort, operating model, and service improvement, not a plugin subscription.

## Procurement Notes

Keep proposals explicit:

- "Plugin license fee: EUR 0" or equivalent local currency.
- "Third-party licenses and tenant access are customer-owned."
- "Implementation, customization, training, and support are scoped separately."
- "AI model usage costs depend on the customer's chosen AI platform."
