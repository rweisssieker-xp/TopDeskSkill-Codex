# Privacy Policy

This local plugin does not collect, transmit, or store data by itself.

Bundled skills, scripts, and templates operate on files and environment variables available in the local workspace. Any connection to a TOPdesk tenant or external API requires explicit local configuration, such as `TOPDESK_BASE_URL` and `TOPDESK_API_TOKEN`.

When using TOPdesk data, treat incident text, caller details, operator notes, branch/customer fields, assets, and AI prompts as sensitive. Apply tenant permissions, PII minimization, audit logging, and retention rules before sharing outputs.
