# TOPdesk Skill Suite Feature Matrix

This feature matrix is the complete US-English marketing and sales catalog for the 48 bundled TOPdesk skills. It is intentionally factual: each entry describes accelerator capabilities, reusable artifacts, scripts, templates, or local runtime functions that ship with the plugin. It does not claim a hosted SaaS product, managed connector, or included third-party license.

## Positioning Summary

TOPdesk Skill Suite is a Service Intelligence Operating System accelerator for TOPdesk work. It combines domain guidance, tenant mapping, Power BI engineering, lifecycle analytics, AI governance, automation safety, data quality, operations, proof-of-value material, and executive evidence into one inspectable plugin.

## Market Differentiation

- Beyond Excel exports: adds lifecycle facts, KPI lineage, reconciliation, Power BI patterns, and action-oriented findings.
- Beyond generic AI prompts: adds TOPdesk-specific governance, evals, adoption evidence, permissions, and review patterns.
- Beyond pure Power BI delivery: adds tenant drift, process debt, AI governance, automation safety, readiness scoring, and operating cadence.
- Beyond classic TOPdesk consulting: packages reusable scripts, templates, scorecards, runbooks, and release evidence.
- Beyond process-mining platforms: provides lightweight TOPdesk-specific what-if and lifecycle analysis without a separate platform rollout.
- Beyond proprietary analytics products: keeps artifacts inspectable in Markdown, CSV, DAX, Power Query, Python, PowerShell, and local runtime files.
