# Page Wireframes

## Executive Overview

1. KPI strip: created, closed, backlog, SLA met %, major incidents.
2. Main trend: created vs closed over time.
3. Breakdown: incidents by category or branch.
4. Exception table: SLA breaches, major incidents, oldest open tickets.

## Operations Queue

1. Filters: date, branch, operator group, priority.
2. KPI strip: open, at risk, breached, unassigned.
3. Matrix: operator group by status/priority.
4. Detail table: ticket, caller, age, SLA due, owner.

## Data Quality

1. KPI strip: missing caller, missing branch, unmapped category, duplicate persons.
2. Trend: findings over time.
3. Breakdown: findings by entity and severity.
4. Action table: finding, owner, suggested fix.
