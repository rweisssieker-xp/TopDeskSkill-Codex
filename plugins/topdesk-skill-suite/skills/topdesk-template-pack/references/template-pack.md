# TOPdesk Template Pack

Use this file when creating or reusing TOPdesk templates.

## Template Categories

- SQL reporting views.
- DAX measures.
- Power Query functions.
- AI prompt JSON schemas.
- Test cases.
- Runbooks.
- Discovery questionnaires.
- Proposal outlines.
- Workshop agendas.
- User story templates.

## Template Selection

- For reporting: use SQL/DAX/Power Query templates.
- For delivery: use epic, story, roadmap, and acceptance templates.
- For operations: use runbook and incident response templates.
- For AI: use structured JSON prompt/output templates.
- For sales: use discovery, USP, and ROI templates.
