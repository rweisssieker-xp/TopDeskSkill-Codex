# TOPdesk Project Delivery

Use this file for epics, user stories, roadmaps, delivery plans, estimates, milestones, and project governance.

## Delivery Streams

- Discovery and tenant mapping.
- Data foundation and schema.
- OData/API integration.
- Power BI reporting.
- Workflow/admin configuration.
- AI/KI assist.
- Security/compliance.
- Testing and rollout.
- Enablement and adoption.

## Roadmap Pattern

Phase 1: Discovery and source verification.
Phase 2: Data model and integration baseline.
Phase 3: Power BI MVP.
Phase 4: Workflow/data-quality improvements.
Phase 5: AI suggest-only pilot.
Phase 6: Controlled automation and continuous improvement.

## Estimation Factors

- Number of TOPdesk modules in scope.
- OData/API availability and metadata quality.
- Data-quality issues.
- RLS/branch/customer complexity.
- Historical backfill volume.
- Number of Power BI pages and measures.
- AI use-case risk and evaluation data availability.
- Change management and training needs.
