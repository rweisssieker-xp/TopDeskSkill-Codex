# TOPdesk Report Factory

Use this file to generate repeatable report specs, DAX backlogs, page plans, validation plans, and implementation tickets.

## Inputs

- Audience and business questions.
- TOPdesk modules and entities.
- KPI definitions.
- OData/API/export availability.
- RLS requirements.
- Existing report screenshots or PBIX docs.

## Output Pack

1. Report specification.
2. Semantic model table list.
3. Relationship list.
4. DAX measure backlog.
5. Page/visual backlog.
6. RLS design.
7. Refresh design.
8. Reconciliation plan.
9. Data-quality checks.
10. Implementation user stories.
