# TOPdesk Proof Of Value

Use this skill to package a short, credible TOPdesk proof-of-value sprint.

## Workflow

1. Pick one measurable value theme: SLA focus, reporting automation, AI assist, data quality, routing, knowledge deflection, or migration confidence.
2. Define required tenant artifacts: OData metadata, sample exports, KPI definitions, process rules, role/security assumptions.
3. Create PoV outputs: Power BI prototype, semantic model sketch, AI feature pack, data-quality findings, ROI calculator, demo script, decision memo.
4. Define success metrics and acceptance criteria before implementation.
5. Include a Power BI page plan and measures for baseline vs target.
6. Produce next-step offer packaging: pilot, foundation, production rollout, governance cockpit.
