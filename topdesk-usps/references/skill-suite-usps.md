# TOPdesk Skill Suite USPs

Use this file when positioning the skill suite itself: what makes the Codex skill system valuable, differentiated, reusable, and credible.

## Meta-USP

A complete TOPdesk delivery and intelligence skill system: tenant-verified, Power BI-ready, AI-governed, auditable, operationalizable, and business-case-ready from discovery to production operations.

## One-Liners

- "A full TOPdesk expert system for schema, OData, Power BI, AI, governance, operations, rollout, and ROI."
- "Not a prompt collection: a validated TOPdesk delivery framework with scripts, templates, handbooks, and release automation."
- "Turns TOPdesk tenant artifacts into mapped schemas, lifecycle history, governed analytics, AI-assist patterns, and operational runbooks."
- "Bridges service desk practice, data engineering, BI, AI governance, and executive value in one reusable skill suite."

## Top Differentiators

1. **39 validated specialized skills**
   - Broad expert skill plus focused skills for BI, AI, OData, schema, security, testing, operations, workflows, handbooks, ROI, tenant mapping, report factory, and more.

2. **Tenant-verification-first**
   - Uses OData `$metadata`, API samples, CSV exports, and actual artifacts as source of truth.
   - Avoids inventing internal TOPdesk database fields.

3. **Power BI depth**
   - KPI catalog, semantic model guidance, DAX snippets, Power Query patterns, RLS, report specs, model review, refresh, and reconciliation.
   - Includes incident lifecycle facts for status duration, assignment duration, daily snapshots, reassignment rate, and bottleneck analysis.

4. **AI/KI with controls**
   - Classification, routing, summaries, draft replies, RAG, duplicate detection, prediction.
   - Includes human review, confidence, audit, PII controls, prompt/model versioning, and feedback loops.

5. **End-to-end delivery**
   - Covers discovery, schema, OData, BI, AI, security, testing, migration, training, operations, release, and business case.

6. **Operational tooling**
   - Metadata parser, CSV profiler, field catalog generator, data-quality findings generator, tenant mapping pipeline, release packager, install script, validation script.

7. **Handbook-ready**
   - Admin, operator, Power BI, AI governance, integration, operations, schema, rollout, training, and executive handbook templates.

8. **Quality gates built in**
   - Skill validation, Python compile, marker checks, reference checks, GitHub Actions, release checklist, release zip, git tag.

9. **Business and technical alignment**
   - Can generate SQL/DAX/Power Query/prompts and also USPs, ROI, discovery questions, roadmap, proposal, and executive narrative.

10. **Reusable accelerators**
    - DAX, Power Query, SQL, test case, runbook, proposal, and user-story templates.

11. **Historical service-flow model**
    - Separates current incident state from transition facts and daily snapshots, so Power BI can explain where incidents waited and how routing changed over time.

## Executive Pitch

This skill suite reduces the cost and risk of TOPdesk improvement programs by packaging the repeatable expertise needed for tenant mapping, database design, lifecycle history, governed Power BI reporting, safe AI assistance, security controls, operational runbooks, training, and ROI justification. It helps teams move faster without losing auditability or tenant-specific accuracy.

## Technical Pitch

The suite combines Codex skills, structured references, scripts, assets, validation, CI, and release tooling. It treats real TOPdesk metadata and exports as authoritative inputs, then guides transformation into field catalogs, schemas, lifecycle history, Power BI models, AI workflows, runbooks, and delivery artifacts.

## Sales Pitch

Instead of starting every TOPdesk analytics or AI project from a blank page, this suite provides a validated accelerator: discovery questions, tenant-mapping pipeline, lifecycle facts, Power BI model patterns, AI governance, data-quality checks, delivery roadmap, handbooks, and business-case material.

## Operator Trust Pitch

The AI parts are designed to help operators, not replace them. Operators review suggestions, see confidence and reasons, keep control over customer-visible communication, and provide feedback that improves future suggestions.

## Security Pitch

The suite assumes service desk data is sensitive. It includes PII minimization, permission-aware retrieval, RLS, audit logging, secret handling, human approval gates, and explicit controls for AI-generated content.

## Maximum Positioning Statement

The TOPdesk Skill Suite is a production-oriented delivery accelerator for TOPdesk intelligence work. It combines domain-specific service-management expertise, tenant-verified schema discovery, lifecycle history, Power BI engineering, AI/KI governance, security controls, operational runbooks, migration guidance, training material, executive value messaging, and release automation into one validated skill ecosystem.
