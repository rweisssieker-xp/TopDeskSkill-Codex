# TOPdesk Handbook Set

This handbook set is the user-facing documentation layer for the TOPdesk skill suite.

## Handbooks

| Handbook | Audience | Purpose |
| --- | --- | --- |
| Admin Handbook | Application managers | Configure categories, statuses, supporting files, forms, templates, and governance. |
| Operator Handbook | Service desk operators | Work incidents, notes, knowledge, AI suggestions, closure, and escalation. |
| Power BI Handbook | BI developers and report consumers | Understand semantic model, KPIs, RLS, refresh, and validation. |
| AI Governance Handbook | Product, security, and service leads | Use AI/KI safely with human review, audit, privacy, and evaluation. |
| Integration Handbook | Developers and integration owners | Design OData/API sync, mapping, idempotency, retries, and reconciliation. |
| Operations Handbook | Operations/support owners | Run deployments, monitoring, refresh, backup, and incident runbooks. |
| Schema Handbook | Data engineers and architects | Understand database model, external IDs, history, views, and migration. |
| Rollout Handbook | Project and change leads | Plan migration, cutover, adoption, training, and success metrics. |
| Training Handbook | Trainers and team leads | Deliver role-based enablement and demos. |
| Executive Handbook | Leaders and sponsors | Understand value, USPs, ROI, roadmap, and governance posture. |

## Source Skill

Use `topdesk-handbook` to generate, update, or tailor these documents.
